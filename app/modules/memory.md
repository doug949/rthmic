# PILLAR: Memory

## USER INTENT
The user wants to remember information by making it feel connected, vivid, and easy to retrieve.

## CORE FUNCTION
Transform loose information into memorable associations through linked scenes, sensory imagery, emotional hooks, and personal references when available.

## EMOTIONAL STATE
Scattered or effortful recall -> vivid recognition and confident retrieval

## SONG BEHAVIOUR
- Structure: a linked sequence of images, scenes, or associations that naturally leads from one item to the next.
- Tone: warm, vivid, and lightly curious.
- Language: concrete, sensory, and image-rich.
- Musical feel: melodic and memorable, with recurring motifs that reinforce connections.

## STRUCTURAL RULES
- Begin by establishing a memorable setting, object, or emotional anchor.
- Connect each idea to the next through association rather than listing.
- Use chains, scenes, locations, transformations, or cause-and-effect links.
- Reinforce the relationships between items more than the items in isolation.
- Return to key associations at natural musical moments.

## LANGUAGE RULES
- Use visual, tactile, spatial, and emotional imagery.
- Use personal references if the user provides them.
- Make the lyric feel like a scene unfolding, not a study note.
- Repeat only the links that help retrieval.
- Keep the wording simple enough to be sung back.

## CONSTRAINTS
- Do not produce a plain list.
- Do not rely on abstract explanation.
- Do not overload the song with too many disconnected images.
- Do not make the song feel like a lesson or memory lecture.
- Do not introduce personal references that the user has not provided.

## SUCCESS CRITERIA
The user can recall the target material by moving through the song's associations, scenes, or emotional anchors.
