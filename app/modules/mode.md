# PILLAR: Mode

## USER INTENT
The user wants help during a negative state, spiral, freeze, overwhelm, panic, shame, anger, or emotional drop.

## CORE FUNCTION
Interrupt the active pattern quickly, acknowledge the state directly, and guide the user toward relief and grounding.

## EMOTIONAL STATE
Interruption -> relief -> stability

## SONG BEHAVIOUR
- Structure: immediate interruption, direct acknowledgement, fast stabilisation, calm grounding.
- Tone: clear, close, and reassuring.
- Language: direct but gentle.
- Musical feel: starts with a pattern break, then settles into a stable pulse.

## STRUCTURAL RULES
- Interrupt the negative loop early.
- Acknowledge the user's current state without arguing with it.
- Reduce intensity before introducing perspective.
- Use short stabilising lines that are easy to receive.
- Transition into grounded, repeatable calm.
- End with steadiness rather than triumph.

## LANGUAGE RULES
- Use present-tense grounding.
- Keep lines short during the interruption phase.
- Validate the state without amplifying it.
- Use simple sensory anchors such as breath, feet, hands, room, sound, or temperature.
- Avoid abstract reassurance until the user is stabilised.

## CONSTRAINTS
- Do not build slowly before interrupting.
- Do not minimise or debate the user's state.
- Do not turn the song into a pep talk.
- Do not introduce a long explanation.
- Do not use menu, memory, movement, or event-preparation behaviours.

## SUCCESS CRITERIA
The user feels a quick reduction in emotional intensity and a clearer path back to steadiness.
