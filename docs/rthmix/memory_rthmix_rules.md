# Memory Rthmix Rules

Memory Rthmix albums are chaptered learning albums. Each track is a small lesson that installs one unlock and connects it to the chain before it.

## Playback Behaviour

- Memory Rthmix tracks loop by default.
- The listener actively moves on when ready, using the normal next-track action in the app, lock screen, or AirPods.
- The end of every learning track reminds the listener there is no hurry: repeat this one until it feels comfortable, then move on.
- Album order matters. Starting from any track creates a queue from that track onward.

## Track Zero

Track zero explains:
- what the album teaches
- how to use the Rthmix format
- that each track contains one unlock
- that each track can be looped as often as needed
- that the listener moves forward only when ready

Track zero does not teach the full album in detail. It sets the method and names the journey.

## Learning Tracks

Each learning track must:
- introduce itself by naming the current unlock
- briefly nod to the previous unlock, if there is one
- connect the previous unlock to the current one conceptually or narratively
- avoid dwelling on the previous unlock
- teach exactly one new unlock
- spell the target word or phrase when language learning is involved
- call out silent letters or unusual pronunciation when relevant
- end by reminding the listener not to move on until comfortable

Example transition for language learning:

```text
You know how to say hello.
Now what do you say when you leave?
```

Midpoint tracks should acknowledge progress naturally:

```text
You are halfway through now.
Three words are already in the chain.
```

## Final Learning Track

The final learning track should feel like a summit:
- acknowledge that this is the last new unlock
- briefly recall the existing chain
- teach the final unlock
- combine the whole chain in a practical scene or story
- keep the tone celebratory, not exam-like

For language learning, the final track should include a short real-world exchange using all learned words.

## Bonus Track

The bonus track is not another unlock. It is a reflection and consolidation track:
- celebrate what has been achieved
- run the whole chain
- place the chain in a memorable practical scene
- give the listener a moment to acknowledge progress

## Language-Specific Rules

- Spell the foreign-language word clearly in the lyrics.
- If letters are silent, say so plainly.
- If pronunciation is likely to be misread by an English speaker, include a simple phonetic cue.
- Use the real word repeatedly, not only the hook.
- Hooks are retrieval handles, not replacements for the word.
