# RTHM Template: Menu

## Purpose
Navigate a decision. Used when someone is stuck in analysis paralysis, has too many options, or is avoiding a choice. The track forces a commit by making the decision process feel contained and finite.

## Tone
Curious but direct. Slightly playful — removes the weight from the choice. Options are presented as neutral, not loaded. The rhythm encourages movement rather than more deliberation.

## Rhythm Style
- BPM: 88–100
- Time signature: 4/4
- Energy arc: builds through the options, peaks at the decision point, drops cleanly at commitment
- Percussion: light, questioning feel — softer snare, less kick dominance
- Texture: melodic elements that suggest possibility without overwhelm

## Output Constraints
- Duration: 90 seconds
- No more than 3–4 options surfaced (even if user described more)
- Must include a clear commitment instruction
- End with forward motion — not reflection

## Lyric Structure
```
[Frame]
Name the stuck state.
What are you choosing between?
1–2 lines.

[Options]
List 2–4 options as clean, parallel phrases.
Equal weight — no bias toward any option.

[Commit]
Force the decision.
One line: "Pick one. Now."
Followed by forward motion instruction.
```

## Keywords to Use
choose, option, pick, decide, one path, either, or, commit, move, which, done deciding, forward, next

## Example Output
```
[Frame]
Two roads. You've been standing here too long.

[Options]
Option one: start small, build up.
Option two: go big, adjust later.
Option three: ask someone who knows.
Option four: do nothing and see.

[Commit]
Pick one. Right now.
You can change it later.
But you have to move first.
Go.
```
