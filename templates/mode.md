# RTHM Template: Mode

## Purpose
Activate a specific performance state. Used when someone needs to enter deep focus, creative flow, execution mode, or peak performance. The user knows what they need to do — they need the system to get them there.

## Tone
Direct. No warmup. Assumes the user is ready to move. Short declarative phrases. No filler.

## Rhythm Style
- BPM: 95–115
- Time signature: 4/4
- Energy arc: starts at 60%, builds to 90% by the midpoint, holds
- Percussion: steady, forward-leaning kick pattern
- Texture: sparse → dense over 30 seconds

## Output Constraints
- Duration: 90 seconds
- No more than 3 distinct lyric sections
- Each section: 4–6 lines max
- Lines should be commands or statements, not questions
- Avoid metaphor overload — keep language functional

## Lyric Structure
```
[Activation]
Short sharp opening — 2–3 lines establishing the mode

[Lock-in]
The core state instruction — 4 lines max
Rhythm-synced phrases

[Hold]
Repetition or affirmation of locked state
2–3 lines, closes the loop
```

## Keywords to Use
focus, lock, clear, now, move, set, build, hold, activate, enter, go, run, execute

## Example Output
```
[Activation]
Eyes forward. Task open. Clock running.
This is the mode. Enter it.

[Lock-in]
Filter everything else.
One thing. One window.
The work is the only signal.
Stay in it.

[Hold]
You're in it now.
Keep the thread.
Don't stop.
```
